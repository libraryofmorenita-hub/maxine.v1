The Mission

Restoring autonomy.
One command at a time.

For a quadriplegic patient, the inability to independently adjust their position is not a minor inconvenience — it is a daily erosion of dignity. Existing hospital bed controls require manual dexterity. Standard call systems require physical button access. For patients with cervical spinal cord injuries, ALS, or advanced neuromuscular conditions, these interfaces are inaccessible by design.

Cura Personalis — care for the whole person — demands better. MAXINE was built in direct partnership with Volunteers for Medical Engineering (VME) to address exactly this clinical gap.

Voice First

Patients issue natural voice commands to control bed position. No buttons. No physical reach. No dependency on caregiver availability.

LCD Feedback Loop

Every command echoed to a 16×2 LCD display and confirmed via speaker. The patient always knows system state. No silent failures.

CALL HELP Interrupt

A hardwired safety interrupt on Channel 7 — accessible even during system hold states. Emergency access is never blocked by system constraints.

Safety architecture layers
Recognition threshold, command validation, noise protocol, CALL HELP interrupt, feedback loop
L1

Recognition Threshold

The VR3 module is configured with a confidence threshold below which commands are rejected rather than approximated. An uncertain command produces no actuation — it is better to ask again than to move incorrectly.

L2

Command Validation

The Arduino validates every received signal against the known command set before routing to any relay channel. Unrecognized signals trigger a hold state and LCD error display. No unintended actuation is possible.

L3

High-Noise Protocol

In environments where ambient noise exceeds reliable recognition thresholds, the system enters a conservative safety hold. The LCD displays the noise state. All bed movement stops. The environment must stabilize before commands resume.

L4

CALL HELP Interrupt

Channel 7 is hardwired as a safety interrupt — reserved exclusively for nurse call. It remains accessible even when all other channels are in hold state, ensuring emergency access is never blocked by system constraints.

L5

Feedback Loop

Every command — successful or failed — produces an LCD display response and speaker confirmation. The patient always knows the system state. No silent failures. No ambiguity about whether the command was received.
